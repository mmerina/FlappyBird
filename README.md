# FlappyBird

# 简介
用canvas完成的简单小游戏：flappybird，利用了面向对象的思想和中介者模式，全部动态效果使用全局定时器实现。

# 项目文件夹：
该项目的文件夹结构如下所示：<br>
```javascript
    ┠ audio 音频文件
    ┠ images 图片文件
	  ┠ js 	开发文件
		┃  ┠	Scene 场景管理
    ┃  ┠	Bird 鸟类
    ┃  ┠	Land 草地类
    ┃  ┠	Game 游戏类
    ┃  ┠	Pipe 管子类
    ┃  ┠	Background 背景类
    ┠	index 主页
		
```
# 项目启动
直接打开index就能玩
